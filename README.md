
AfterCredist is a full-stack movie platform that combines a Java Spring Boot backend with a React/Vite frontend. The app is designed for browsing movies, managing personal watchlists, and sharing ratings and reviews through a polished, responsive interface.

Key Features:

- Secure user authentication and registration with JWT
- Movie browsing and detailed movie information pages
- User reviews and movie ratings
- Personalized watchlist creation and management
- Responsive React frontend with fast navigation and search
- Spring Boot REST APIs with persistent data storage
